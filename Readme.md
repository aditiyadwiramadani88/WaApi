# API WhatsApp Non Official 
## Feature 
- User Management 
- Mutiple Device 
- Webhook 
- Send Message
- Send Media Message 

## Installation 
- Install nodejs and mysql 
- Create database and configuration database in file config.json

## Install dependence 
```sh
npm install 
```

## Migrate Database

```sh
npx sequelize-cli db:migrate
```
## Seed Database 
```sh 
npx sequelize-cli db:seed:all
```
